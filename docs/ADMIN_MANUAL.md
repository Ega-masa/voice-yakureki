# 音声薬歴ツール 管理者マニュアル

## 1. ログイン

管理者IDとパスワードでログインします。

1. `https://voice-yakureki.vercel.app` を開く
2. 「店舗ID / 管理者ID」に管理者ID（例: `admin-masa`）を入力
3. パスワードを入力
4. ログインボタンをクリック → 管理画面に直接遷移

## 2. 店舗の追加

1. 管理画面の「店舗管理」タブを開く
2. 「新規追加」ボタンをクリック
3. 以下を入力：
   - 店舗名（漢字）*必須*
   - 店舗名（フリガナ）
   - 所属会社 *必須*
   - 最大ユーザー数
   - メモ
4. 「店舗パスワード初期設定」欄で：
   - 「自動生成」ボタンをクリック → パスワードが表示される
   - **パスワードをメモする**（画面を閉じると見えなくなります）
5. 「追加」ボタンをクリック
6. 店舗ID（例: `YK-A3B7X2`）が自動生成される

→ 店舗IDとパスワードを店舗スタッフに共有してください

## 3. 既存店舗のパスワード設定

SQL移行後、既存店舗にはIDが自動付与されていますが、パスワードが未設定です。

1. 店舗一覧で「未開通」と表示されている店舗の編集ボタン（✏️）をクリック
2. 「パスワード初期設定」欄で「自動生成」→「設定」をクリック
3. `✅ ログインアカウントを作成しました` と表示されたら完了

## 4. パスワードの変更

1. 店舗の編集ボタンをクリック
2. 「パスワードを変更する」ボタンをクリック
3. 新しいパスワードを入力（または「自動生成」）
4. 「変更」ボタンをクリック

## 5. 店舗一覧のCSV出力

1. 店舗管理タブのヘッダーにある「CSV」ボタンをクリック
2. 店舗名・店舗ID・会社名・ステータスを含むCSVがダウンロードされる

## 6. 管理者アカウントの追加

1. 「ユーザー」タブの「管理者追加」ボタンをクリック
2. 以下を入力：
   - 管理者ID（例: `admin-tanaka`）
   - 表示名
   - パスワード
   - ロール（システム管理者 or 会社管理者）
   - 所属会社（会社管理者の場合）
3. 「作成」をクリック

## 7. ロール権限の階層

ロール管理タブで設定。上にあるロールほど権限が高い。

| ロール | できること |
|--------|----------|
| 全体管理者（sort_order: 1） | 全機能。全会社の管理 |
| 店舗管理者（sort_order: 2） | 自社の店舗・ユーザー管理 |
| 薬剤師（sort_order: 3） | 録音・閲覧のみ |

下位ロールは上位ロールのユーザーを閲覧・操作できません。

## 8. Slack通知の設定

1. 管理画面「通知」タブを開く
2. 「新規追加」をクリック
3. Slack Incoming Webhook URLを入力
4. 通知したいイベントにチェック：
   - 新規登録申請
   - パスワードリセット依頼
   - ユーザー承認
   - 日次統計レポート
5. 「テスト」ボタンで動作確認

Webhook URLの取得方法：
Slack App → Incoming Webhooks → Add New Webhook → チャンネル選択 → URLコピー

## 9. セッション管理

- 8時間無操作で自動ログアウト
- クリック・キー入力・タッチで自動延長
- 店舗共有PCでの使用を想定

## 10. システム診断

管理画面「診断」タブで以下を確認できます：
- Supabase接続状態
- SOAP API（Claude Haiku）の動作状態
- 環境変数の設定状態（未設定があれば表示）
- DB統計（レコード数・ユーザー数・店舗数・会社数）

## 11. トラブルシューティング

| 問題 | 原因 | 対処 |
|------|------|------|
| ログインできない | IDまたはパスワードが違う | 管理者にパスワードリセットを依頼 |
| 録音後にGroq APIエラー | APIキーが未設定 | 管理画面 → API設定でGroqキーを登録 |
| 診断でSERVICE_ROLE_KEY: false | Vercel環境変数が未設定 | Vercel → Settings → Environment Variables で設定後Redeploy |
| 店舗IDが表示されない | login_migrationが未実行 | SQL Editorで login_migration.sql を実行 |

## 12. Vercel環境変数一覧

| 変数名 | 必須 | 用途 |
|--------|------|------|
| SUPABASE_URL | ✅ | Supabase接続URL |
| SUPABASE_ANON_KEY | ✅ | Supabaseクライアント認証 |
| SUPABASE_SERVICE_ROLE_KEY | ✅ | アカウント作成・パスワードリセット |
| ANTHROPIC_API_KEY | ✅ | AI SOAP分類 |
